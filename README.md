# IoT Blockchain Authentication & Data Logging

A blockchain-based smart contract for secure authentication and trusted data recording 
in IoT networks. Built using Solidity, Hardhat, and Remix.

## Features
- Device registration by admin
- Secure device authentication using nonces
- Replay-attack prevention
- Tamper-proof data logging (hashed records)
- Event-based transparency

## Tech Stack
- Solidity 0.8.x
- Hardhat
- Node.js
- Ethers.js / Web3.js
- Remix (for quick testing)

## Contract Location
`contracts/IoTAuthLogging.sol`
